BUKU TAMU

SELAMAT DATANG

Saya ucapkan selamat datang dan terima kasih karena sudah berkunjung di website pribadi saya. Nama saya Lovia dari kelas X SIJA 1. Di sini saya menampilkan sedikit informasi tentang saya, hobi, dan hal yang saya suka. Selamat Berkunjung!

Saya sudah hobi menggambar sejak kecil. Saya rasa itu karena menggambar dapat mengalihkan pehatian saya pada hal yang saya gambar. Saya Menggambar dengan media digital dan mannual. Saya menekuni menggambar karena saya merasa kurang puas jika hanya menggambar hal-hal dasarnya.

Sejak kecil saya suka untuk menggambar hal-hal disekitar saya, skenario tertentu, dan mengikuti imajinasi saya. Saya menggambar menggunakan pensil dan pulpen. Dahulu saya sangat rajin menggambar hingga ibu saya kadang menanyai jika saya tidak menggambar. Saat saya kecil gambaran saya sudah pasti tidak sebagus sekarang.

Saya menggambar manual saat saya masih kecil dan jika ada tugas menggambar, selain itu saya hanya menggambar manual jika saya bosan saja. Mengambar manual memang tidak se-simpel menggambar digital. Tetapi, dengan menggammbar manual saya jadi tahu dasar-dasar dalam menggambar dan memiliki pengalaman yang lebih terasa. Terakhir saya menggambar manual hingga tuntas adalah saat saya kelas 9 karena, ada tugas untuk menggambar. Tugas menggambar yang diberikan bervariasi dan menyenangkan. Saya suka feeling menggambar manual yang cukup satisfying.

Saya mulai menggambar digital sejak akhir kelas 7 SMP. Kala itu saya sempat pesimis untuk menggambar digital karena belum terbiasa. Namun, saya memberi kesempatan kedua untuk menggambar digital dan saya meneruskannya hingga saat ini. Jika dibandingkan dengan menggambar manual, menurut saya memang lebih simpel dan hemat alat. Tetapi, prosesnya lebih terasa saat menggambar manual. Saya sering menggambar potrait, menggunakan referensi, maupun tanpa refensi.
